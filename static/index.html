문제 세트 가져오기

NotebookLM이나 외부 AI가 만든 JSON 문제 세트를 붙여넣으면 현재 카드와 매칭해 모의 세트로 불러옵니다.

지원 형식: JSON 배열 또는 {"questions": [...]}. 각 문항에는 최소 question_type, prompt, 그리고 현재 카드와 매칭될 card_id 또는 concept_term/term이 필요합니다. 한은형 세트라면 session_mode, section, points, expected_time_minutes, answer_guide도 함께 넣을 수 있습니다.
